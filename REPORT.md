# Web Development Assignment Report

**Project**: Trinidad and Tobago Food Website
**GitHub Repository**: https://github.com/Siahhhisaqt/webdevtrinifood

## Project Overview
A 6-page responsive website showcasing Trinidadian cuisine, built with **HTML5**, **Tailwind CSS**, and **Vanilla JavaScript**. The site features a custom design, mobile-responsive navigation, and an interactive image gallery.

## Technology Stack
-   **HTML5**: Semantic structure (`<nav>`, `<main>`, `<article>`, `<footer>`).
-   **CSS**: Tailwind CSS (v3.4) for styling and responsiveness.
-   **JavaScript**: Custom scripts for Mobile Menu, Lightbox, and Form Validation.
-   **Assets**: Google Fonts (Poppins, Open Sans) and optimized images.

## Key Features
1.  **Responsive Design**: Adapts to Mobile, Tablet, and Desktop screens.
2.  **Interactive Gallery**: Custom JavaScript lightbox with lazy loading.
3.  **Contact Form**: Validates Name, Email, Phone, and Address client-side.
4.  **Accessibility**: ARIA attributes (`aria-expanded`) and semantic HTML.

## Performance & SEO
-   **Lighthouse Scores**: Optimized for Performance, Accessibility, Best Practices, and SEO.
-   **Optimization**: Lazy loading (`loading="lazy"`) and local asset serving.

## Deployment
-   **Run Locally**: `npm install` -> `npm run build:css` -> Open `index.html`.
-   **Live Deployment**: Ready for Netlify/Vercel (Build command: `npm run build:css`).
